기준 div

padding

테두리와 컨텐츠 영역 사이의 거리를 조절하는 속성
padding: 100px;
콘텐츠 영역
padding-top: 50px; padding-left: 100px;
콘텐츠 영역

margin

box 레벨(block)이나 inline 레벨의 요소들 간의 간격을 조절하는 속성

margin: 50px;
콘텐츠 영역
margin-top: 50px; margin-left: 50px;
콘텐츠 영역
margin: auto;
콘텐츠 영역
