화면 배치(표시) 방법 변경 - display

블럭 요소와 인라인 요소의 속성을 변경하여 배치 방식을 변경할 수 있다.

블럭 요소를 인라인 요소로 변경

>> display: inline 테스트
        인라인 요소로 변경되면서 width와 height 속성이 무시된다.
        - inline(줄 내부)에 표기한다 == 글자 처럼 취급
        - width와 height는 내용(content) 영역의 크기를 설정하는 속성
        -> inline 형식의 요소는 width와 height 속성이 적용되지 않는다.

블럭 요소를 인라인 블럭 요소로 변경

>> display: inline-block 테스트
            영역은 인라인 요소로 변경(가로 배치)되지만
    내용은 블럭 요소가 되어 width와 height 속성을 설정할 수 있다.
    (요소간에 작은 간격이 존재)

인라인 요소를 블럭 요소로 변경

>> display: block 테스트
    인라인 요소를 블럭 요소로 변경하면 width와 height값을 지정할 수 있고
    세로로 쌓인다.
